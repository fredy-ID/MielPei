## KNP_SNAPPY

Si vous êtessur windows, knp_snappy devrait fonctionner sans problème puisque l'outil est intégré à l'application. Sinon si vous êtes sur d'autres système d'exploitation, il vous faudra installer par ligne de commande knp_snappy depuis le site officiel et dé-commenter la ligne correspondante dans <code>app/config/packages/knp_snappy.yaml</code>
