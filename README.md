## KNP_SNAPPY

Si vous êtes sur windows, knp_snappy devrait fonctionner sans problème car puisque l'outil est directement intégré à l'application. Sinon si vous êtes sur un autre système d'exploitation, il vous faudra installer par ligne de commande knp_snappy depuis le site officiel et dé-commenter la ligne correspondante dans <code> app/config/packages/knp_snappy.yaml </code>
